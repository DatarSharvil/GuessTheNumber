# About:
  A CLI game that selects a number between 1 and 100 and you have to chose it! Works on Linux x64 (Debian based) without an internet. Enjoy!
# Requirements:
  <p> Debian Based x64 OS (Ubuntu, Debian, Mint, Pop!_OS, Zorin, etc.)
  <p> Minimum: Java 21 </p>
  <p> Recommended: Java 21 </p>

# Usage:
  1. Linux (Debian-based): Just install the .deb file the way you like and enter the command "guessthenumber" from any terminal to execute.
